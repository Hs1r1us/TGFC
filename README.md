# TGFC·NG

**TGFC论坛非官方客户端**

此项目为HiPDA 绿皮怪兽版的一个分支，
为解决TGFC WAP版加载速度过慢开发的一个TGFC专用版本客户端，数据源为WEB版本TGFC。


多谢原作者 绿皮怪兽GreenSkinMonster， jejer 的分享
